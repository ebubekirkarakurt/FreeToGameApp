## Installation

Install my-project with npm

```bash
  git clone https://github.com/ebubekirkarakurt/FreeToGameApp.git
  cd FreeToGameApp
```
Install NPM packages
 
```bash
  npm install
```
Start metro server
 
```bash
  npx react-native run-android
  or
  npx react-native run-ios
```
